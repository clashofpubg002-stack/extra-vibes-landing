# ✦ Extra Vibes - Premium Luxury Perfume E-Commerce Landing Page

A sophisticated, high-end e-commerce landing page for a luxury perfume brand, featuring a modern dark mode aesthetic with gold accents. Built with vanilla HTML5, CSS3, and JavaScript—no frameworks required.

## 🌟 Features

### Design & Aesthetic
- **Dark Mode Premium**: Deep obsidian backgrounds with metallic gold accents
- **Luxury Typography**: Playfair Display for headings, Montserrat for body text
- **Responsive Grid**: 4-column layout that adapts to all screen sizes
- **Smooth Animations**: 10+ keyframe animations for a polished feel

### Functionality
- **Shopping Cart**: Fully functional with local storage persistence
- **Product Management**: 8 premium perfume products with detailed information
- **Newsletter Signup**: Email validation and subscription handling
- **Mobile Menu**: Responsive hamburger navigation
- **Toast Notifications**: Real-time feedback for user actions

### Performance & Accessibility
- **100% Responsive**: Mobile, tablet, and desktop optimized
- **Accessibility**: WCAG AA compliant with proper ARIA labels
- **Fast Loading**: Optimized CSS and lazy-loaded images
- **SEO Friendly**: Semantic HTML5 and proper meta tags
- **No Dependencies**: Vanilla JavaScript—no frameworks or libraries
